# Spotify TUI Utility

Пацанская хуйня для Spotify: показывает ваши **liked tracks**, **top tracks**, **top artists**, **прослушанные минуты** и **топ жанров**.  

## Функции

- Dashboard с любимыми треками и артистами (с обложками)  
- Top Tracks / Top Artists с выбором периода (`4 недели`, `6 месяцев`, `All time`)  
- Расчёт минут прослушивания последних треков  
- Топ жанров с прогресс-барами  
- Простая ASCII-анимация в блоке жанров  
- Вся информация получается через **Spotify OAuth (PKCE)**
